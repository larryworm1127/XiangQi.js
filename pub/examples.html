Basic Setup

First, download the library files and unzip the files to your desired location.

Since XiangQi.js uses no external API, to use the XiangQi.js API, simply load the JavaScript file and CSS file into the web page: 

<script type="text/javascript" src="path-to-xiangqi.js"></script>
<link rel="stylesheet" href="path-to-xiangqi.css">

Make sure to also include the assets folder that contains various content used by XiangQi.js under the same directory as the library files. And that should be it! Read the API Doc to see various configurations and API functions that XiangQi.js provides, or read below to see examples of API usage.

Empty Board

XiangQi.js can initialize an empty XiangQi board with just a containerId property in config. 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board'
});

Start Positions

Full start position can be displayed by passing in 'start' for boardContent field 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: 'start'
});

FEN String Defined Board

A XiangQi board with use-defined board content. 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: '1r2g4/9/1h2ea3/4s4/1R3S3/7H1/s3C2cr/3A5/4G4/9'
});

Multiple Boards 

// HTML
<div id="board1"></div>
<div id="board2"></div>

// JavaScript
const board1 = new XiangQi({
  containerId: 'board1',
  boardContent: 'start',
  boardSize: 350
});
const board2 = new XiangQi({
  containerId: 'board2',
  boardContent: '1r2g4/9/1h2ea3/4s4/1R3S3/7H1/s3C2cr/3A5/4G4/9',
  boardSize: 350
});

// CSS
#board1, #board2 {
    display: inline-block;
    margin-right: 5pt;
}

Show Side Bar Config

Side bar showing piece count of current board can be displayed using showSideBar: true in config. 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: '1r2g4/9/1h2ea3/4s4/1R3S3/7H1/s3C2cr/3A5/4G4/9',
  showSideBar: true
});

Draggable Config

Board pieces can be dragged and dropped around the board by setting draggable: true in config. 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: '1r2g4/9/1h2ea3/4s4/1R3S3/7H1/s3C2cr/3A5/4G4/9',
  draggable: true
});

Clickable Config

Board pieces can be clicked, which will highlight the piece itself and its possible moves. The moves can be clicked to move the piece to that location. This can be enabled by setting clickable: true in config. 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: 'start',
  clickable: true
});

Void Pieces Config

Setting voidPieces: true will allow clickable pieces to highlight moves that can void non-friendly pieces. 

// HTML
<div id="board"></div>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: 'start',
  clickable: true,
  voidPieces: true
});

Flip Board

The orientation of the XiangQi board can be flipped by calling flipBoard() API function. 

// HTML
<div id="board"></div>
<button type="button" id="flip">Flip</button>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: 'start',
});
document.getElementById('flip').addEventListener('click', () => board.flipBoard());

Move Piece

Move objects allow user to define a chain of moves to be displayed sequentially. Each move can be executed using movePiece() API function Below is a few moves from a famous XiangQi opening. 

// HTML
<div id="board"></div>
<button type="button" id="move">Move</button>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: 'start'
});

const generator = getMove();  // getMove() is a generator that produces move objects
document.getElementById('move').onclick = function () {
  const nextMove = generator.next();
  if (!nextMove.done) {
    board.movePiece(nextMove.value);
  } else {
    document.getElementById('move').setAttribute('disabled', 'true');
  }
};

Add Pieces / Remove Pieces

User can add single piece at a time, as well as remove specific piece from board.
Note: below input fields expects valid inputs, otherwise error will be thrown. 



// HTML
<div id="board"></div>
<button type="button" id="addButton">Add Piece</button>
<button type="button" id="removeButton">Remove Piece</button>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: 'start'
});

// addRow, addCol, pieceType, side, removeRow, removeCol are all input fields
addButton.onclick = (e) => {
  e.preventDefault();
  board.drawPiece(parseInt(addRow.value), parseInt(addCol.value), pieceType.value, side.value);
};

removeButton.onclick = (e) => {
  e.preventDefault();
  board.removePiece(parseInt(removeRow.value), parseInt(removeCol.value));
};

Clear / Draw Board

Board content can be cleared using clearBoard() function, and if user wishes to retain current board state after clear, simply pass in false as argument for clear function. The stored board content can be drawn back using drawBoardContent(). If user wish to override current board content, then they can pass their own board content into drawBoardContent() or call drawStartPositions() to draw out start position. 

// HTML
<div id="board"></div>
<button type="button" id="clear">Clear Board</button>
<button type="button" id="draw">Draw Board</button>

// JavaScript
const board = new XiangQi({
  containerId: 'board',
  boardContent: '1r2g4/9/1h2ea3/4s4/1R3S3/7H1/s3C2cr/3A5/4G4/9'
});
document.getElementById('clear').addEventListener('click', () => board.clearBoard(false))
document.getElementById('draw').addEventListener('click', () => board.drawBoardContent())

Resize

The size of the board can be dynamically changed using resizeBoard() API function.

Size: 400px 

// HTML
<div id="board"></div>
<input type="range" min="250" max="750" id="slider">

// JavaScript
const board = new XiangQi({ containerId: 'board', boardContent: 'start' });
document.getElementById('slider').addEventListener('change', event => {
  board.resizeBoard(parseInt(event.target.value));
});
