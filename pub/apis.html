Initial Config options

Note that most of the configurations can be changed later
Property Default Type Description
boardSize 400 Integer The width of the board to be displayed on the web page. The height of the board as well as the size of each piece will be calculated based off this value.
containerId None String The ID of the container for which the board will be placed in. Defaulted to top level document.body if no ID given.
boardContent [] 2D-array of objects or
String		 The content of the board to be displayed.
Possible values for this properties are:
  • 2D-array of Piece object.
  • A FEN string
  • 'start' to have board in start position
showSideBar false boolean Setting showSideBar: true will draw a panel on the side of the board showing the number of pieces on the current board upon calling the API.
draggable false boolean Setting draggable: true will allow each pieces on the board to be dragged around and dropped in an empty square on the board.

Note that this option will be overridden if clickable: true.
delayDraw false boolean Setting delayDraw: true will delay the creation of the XiangQi board on the web page. This allows user to create interactive components where the game board is drawn upon certain external trigger.
redOnBottom false boolean redOnBottom determines the orientation of the board. Setting it to true will cause newly generated start position board to have red side on the bottom of the board rather than on top.

Note that this option ONLY works when boardContent: 'start'.
clickable false boolean Setting clickable: true will allow the generated board pieces to be clickable. Clicking specific pieces on the board will highlight the piece as well as all its possible moves. The user can either click the same piece again to remove the highlighting, or to click one of the highlighted squares to move the piece to that location.
voidPieces false boolean Setting voidPieces: true will enable clickable feature to show moves that can remove non-friendly pieces, which also allows user to click on those moves and void the pieces. If side bar is also shown, then voiding pieces will automatically update piece counts.

API Functions

Function Arguments Description
drawBoard() none Create an empty XiangQi board in given container element.
removeBoard() none Remove the XiangQi board from the given container element.
flipBoard() none Flip the orientation of the board. (Flip red and black sides)
resizeBoard() newWidth Re-size the current XiangQi board to given <newWidth>.
If side bar is present, it will be resized to the same ratio.
movePiece() moveString Update the XiangQi board state using given <moveString> and draw out the updated board content. The function will do nothing is either the <moveString> is invalid or if the move is invalid.
drawStartPositions() none Set the XiangQi board current state to start positions and draw it out on the XiangQi board.
drawBoardContent() boardContent The function will clear the board and draw out current board content. If <boardContent> is provided, then the function will update the current board content with given one and draw it out on the XiangQi board.
clearBoard() clearVirtual Removes clear all pieces from both the physical XiangQi board as well as virtual XiangQi board. If <clearVirtual> is set to false, then the clear will only reflect on the DOM.
drawPiece() row, col, piece, side Draw out given <piece> at position (<row>, <col>). If the position is already occupied, then the function will do nothing.
removePiece() row, col Remove the piece from position (<row>, <col>). If the position is empty, then the function will do nothing.
drawSideBar() none Add the side panel showing piece counts for current state of the board to the DOM. If a side bar is already present, then the function will update the side bar content.
removeSideBar() none Removes the side bar from the DOM. If there is no side bar, then the function will do nothing.
enableVoidPieces() none Enables void pieces such that clickable will be allowed to void non-friendly pieces.
disableVoidPieces() none Disables void pieces such that clickable will not be allowed to void any pieces.
makeDraggable() none Makes all pieces on current XiangQi board draggable. makeDraggable will disable clickable feature upon call.
removeDraggable() none Removes draggable feature for all pieces on the board.
makeClickable() none Makes all pieces on current XiangQi board clickable. makeDraggable will disable draggable feature upon call.
removeClickable() none Removes clickable feature for all pieces on the board.
